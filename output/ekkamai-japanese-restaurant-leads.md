## Website observations

- หน้าเว็บมีข้อมูลครบ แต่โครงสร้างดูเป็น branch information page มากกว่า landing page สำหรับปิดการจอง/การโทร
- CTA โทร/จอง/แผนที่ยังสามารถทำให้เด่นขึ้นมากบนมือถือ
- Hero section ยังมีโอกาสทำให้ดึงดูดขึ้นด้วย positioning: Osaka izakaya, late-night dining, karaoke/private rooms
- ควรเพิ่ม trust/value blocks เช่น เปิดดึก, เมนูเยอะ, ห้องคาราโอเกะ, เดินทางจาก BTS Ekkamai
- เหมาะทำ demo แบบ modern izakaya landing page เพื่อเสนอให้เจ้าของเห็นภาพก่อน/หลัง

## Compliance notes

- ใช้เฉพาะข้อมูล public จาก official website/directory/search results
- ยังไม่ส่งข้อความหาธุรกิจ
- Demo ต้องมี disclaimer ว่าไม่ใช่เว็บทางการ
- ไม่ใช้โลโก้หรือรูปจริงของร้านใน demo นี้ ใช้ gradient/placeholder visual แทน
