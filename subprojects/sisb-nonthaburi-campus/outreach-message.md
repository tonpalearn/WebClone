# Outreach Draft: SISB Nonthaburi Campus

## Short Thai first message

สวัสดีครับ ผมทำตัวอย่างหน้าเว็บ redesign ให้ SISB Nonthaburi Campus แบบดูเร็ว ๆ แล้วเห็นภาพเลยว่า “ข้อมูลเดิมของเว็บยังอยู่ครบ แต่จัดใหม่ให้ดูพรีเมียมและติดต่อสะดวกขึ้นบนมือถือ” ครับ

ผมยังไม่ได้เผยแพร่เป็นเว็บทางการหรืออ้างว่าเกี่ยวข้องกับธุรกิจนะครับ เป็น concept demo สำหรับคุยแนวทางเท่านั้น ถ้าสะดวกผมส่งลิงก์ตัวอย่างให้ดูได้ครับ

## 30-second call script

ผมเห็นว่า SISB Nonthaburi Campus เป็นธุรกิจที่ลูกค้าตัดสินใจจากความน่าเชื่อถือและภาพลักษณ์มาก เลยลองจัดข้อมูล public เดิมใหม่ให้หน้าแรกดูชัดขึ้น — ปุ่มติดต่อเด่นขึ้น, mobile อ่านง่ายขึ้น, และทำให้บริการหลักถูก scan ได้เร็วขึ้น ถ้าถูกทิศทาง ผมสามารถทำเป็นเว็บจริงโดยใช้ content/รูปที่ธุรกิจอนุมัติได้ครับ

## Value proposition bullets

- ใช้ข้อมูลจริงจากเว็บ/แหล่ง public เดิม ไม่แต่ง claim เพิ่ม
- ทำให้ภาพลักษณ์ดู premium ขึ้นโดยไม่หลุด category
- ปรับ mobile CTA ให้โทร/ทัก/จองง่ายขึ้น
- เหมาะกับธุรกิจที่ลูกค้าต้องรู้สึกมั่นใจก่อนติดต่อ
