# Outreach Package: Phed Phed (เผ็ดเผ็ด) — สาขาราชพฤกษ์

## Short Thai First Message

สวัสดีครับ ทีมงาน Phed Phed (เผ็ดเผ็ด) สาขาราชพฤกษ์ครับ ผมชื่นชอบคอนเซ็ปต์ร้านอาหารไทยรสจัดวัตถุดิบ homemade ของทางร้านมากครับ พอดีผมลองทำตัวอย่างหน้าเว็บเวอร์ชันใหม่ของสาขาราชพฤกษ์ไว้ดูเป็นคอนเซ็ปต์ เน้นให้ดูทันสมัย น่ากิน ใช้ง่ายบนมือถือ และมีปุ่มโทร/แอด Line/สั่งเดลิเวอรีที่ชัดเจนขึ้นครับ

ผมทำเป็น "หน้าเปรียบเทียบเว็บเดิมกับเว็บใหม่" ไว้ให้เห็นภาพง่าย ๆ ถ้าสะดวกผมส่งลิงก์ตัวอย่างให้ดูได้ครับ ถ้าชอบค่อยคุยต่อเรื่องปรับเป็นเว็บจริงของร้านได้เลยครับ (เป็นแค่คอนเซ็ปต์ ยังไม่ใช่เว็บทางการนะครับ)

## 30-Second Call Script

สวัสดีครับ ผมชื่อ [ชื่อผู้ติดต่อ] นะครับ ทำงานด้านออกแบบเว็บไซต์ให้ร้านอาหารและธุรกิจท้องถิ่นครับ พอดีผมแวะดูเว็บของ Phed Phed แล้วเห็นว่าตัวร้านน่ากินมาก แต่หน้าเว็บปัจจุบันยังเป็นดีไซน์เก่า ดูบนมือถือลำบาก และข้อมูลหลายสาขาปนกันอยู่ ทำให้คนที่อยากมาสาขาราชพฤกษ์หาข้อมูลยากนิดนึงครับ

ผมเลยลองทำตัวอย่างหน้าเว็บใหม่เฉพาะสาขาราชพฤกษ์ไว้ให้ดูฟรี เป็นหน้าเปรียบเทียบเว็บเดิมกับเว็บใหม่ กดโทร แอด Line หรือสั่งเดลิเวอรีได้ในแตะเดียว ถ้าสนใจผมส่งลิงก์ให้ดูได้เลยครับ

## Value Proposition Bullets

- หน้าเว็บดูน่าเชื่อถือและน่ากินขึ้นภายในไม่กี่วินาทีแรก
- ลูกค้าบนมือถือกดโทร / แอด Line / สั่งเดลิเวอรีได้ใน 1 แตะ
- แยกข้อมูล "สาขาราชพฤกษ์" ให้ชัด แก้ปัญหาข้อมูลหลายสาขาปนกัน
- เหมาะกับการยิงโฆษณาหรือส่งลิงก์ให้ลูกค้าโดยตรง
- หน้าเปรียบเทียบ before/after ช่วยให้เห็นความต่างได้ทันทีในนัดเดียว

## Follow-Up Message

สวัสดีครับ ขออนุญาต follow up เรื่องตัวอย่างหน้าเว็บของสาขาราชพฤกษ์นะครับ ตัวอย่างที่ทำไว้เป็นแค่คอนเซ็ปต์เพื่อใช้พูดคุย ยังไม่ใช่เว็บทางการของร้านครับ หากทางร้านสนใจ อยากปรับสี ข้อความ รูปอาหารจริง หรือเพิ่มเมนู/โปรโมชัน ผมช่วยทำต่อให้เป็นเว็บจริงได้ครับ ไม่มีข้อผูกมัดใด ๆ ครับ
