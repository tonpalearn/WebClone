# Redesign Brief: Phed Phed (เผ็ดเผ็ด) — สาขาราชพฤกษ์

## Objective

สร้างคอนเซ็ปต์ redesign แบบ mobile-first เฉพาะ **สาขาราชพฤกษ์ (The Circle ราชพฤกษ์)** เพื่อแสดงให้เจ้าของเห็นว่าหน้าเว็บที่ชัดและน่ากินกว่าเดิม จะช่วยเพิ่มการโทร จองโต๊ะ และสั่งเดลิเวอรีได้อย่างไร โดยหัวใจของงานคือ **"หน้าเปรียบเทียบเว็บเดิม ↔ เว็บใหม่"** เป็นหน้าแรกสำหรับใช้เสนอขายงานออกแบบ

## Target Audience

- ลูกค้าท้องถิ่นย่านราชพฤกษ์/ชัยพฤกษ์/ราชพฤกษ์-แคราย ที่มองหาร้านอาหารไทยรสจัด
- คนค้นหาร้านบนมือถือ ตัดสินใจเร็วจากภาพและปุ่มกด
- คนที่อยากสั่งเดลิเวอรีหรือโทรจองโต๊ะที่ The Circle ราชพฤกษ์โดยเฉพาะ

## Current Problems Observed

1. ดีไซน์ยุคต้น 2010 หน้าตาเก่า ไม่สื่อความน่ากินของอาหารไทยรสจัด แบรนด์ดูจืด
2. ประสบการณ์บนมือถือแย่ — เลย์เอาต์ไม่ปรับตามจอ ปุ่ม/ตัวอักษรเล็ก โหลดช้า
3. ข้อมูล 8+ สาขายัดรวมในหน้าเดียวไม่จัดระเบียบ หาข้อมูลเฉพาะสาขาราชพฤกษ์ยาก + แบรนด์ย่อยสับสน
4. ปุ่ม Call-to-Action อ่อน ไม่มีปุ่มโทร/จอง/สั่งเดลิเวอรีที่เด่นชัด

## Proposed Improvements

1. Hero ที่ขายบรรยากาศ "ไทยรสจัด homemade" พร้อมพาดหัวชัดและปุ่มหลักเด่น
2. ปุ่มโทร / Line / สั่งเดลิเวอรี ติดมือบนมือถือ กดได้ใน 1 แตะ
3. โฟกัสสาขาราชพฤกษ์เป็นพระเอก แยกข้อมูลสาขาให้ชัด + ลิงก์เล็ก ๆ ไปสาขาอื่น
4. เมนูเด่นแบบ story card สแกนง่ายบนมือถือ (placeholder รูปอาหารด้วย gradient/ไอคอน)
5. เวลาทำการ + แผนที่/ติดต่อ The Circle ราชพฤกษ์ ครบในที่เดียว
6. เลย์เอาต์ mobile-first ทันสมัย โหลดเร็ว

## Page Sections

- **index.html — หน้าเปรียบเทียบ (COMPARISON) เว็บเดิม↔เว็บใหม่ [หน้าหลัก]**: Hero "เว็บเดิม vs เว็บใหม่", กรอบ before/after คู่ (iframe เก่า/ใหม่), ตารางเทียบ 5 ข้อ, CTA เปิดเว็บใหม่/ดูเว็บเดิม, กล่องเสนอขาย + ปุ่มติดต่อทีมออกแบบ, footer + disclaimer
- **new.html — เว็บใหม่เต็มรูปแบบ**: Hero + ปุ่มโทร/Line/เดลิเวอรี, เมนูเด่น, จุดเด่นร้าน (รสจัด homemade), โฟกัสสาขาราชพฤกษ์ + ลิงก์สาขาอื่น, เวลาทำการ, แผนที่/ติดต่อ, footer
- **old.html — หน้าจำลองเว็บเดิม**: สื่อความเก่ายุค 2010 / ข้อมูลหลายสาขารก / CTA อ่อน / ไม่ responsive — มีป้าย "จำลองจากเว็บเดิม • ไม่ใช่ภาพจริง"

## Style Direction

- Mood: สดใส มีชีวิตชีวา น่ากิน — อาหารไทยรสจัด ร้อนแรงแต่ดู premium
- Colors: แดงพริก (#e11d2a / #ff3b30) + ส้มไฟ (#ff7a18 / #ff9e2c) + เขียวพริก/ใบโหระพา (#2fae66) ตัดพื้นดำเข้ม (#0c0705) โทน glass
- Typography: หัวเรื่องตัวหนาคมโมเดิร์น (display serif/sans หนา) + เนื้อหา sans-serif อ่านง่าย
- Imagery: placeholder อาหารด้วย gradient ร้อน + ไอคอน (พริก/ชาม/ไฟ) — ไม่ใช้รูปจริง
- CTA tone: ชวนกินแบบอบอุ่นไม่กดดัน — "โทรจองโต๊ะ", "สั่งเดลิเวอรี", "แอด Line"
