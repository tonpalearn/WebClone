# Redesign Brief: SHAKARIKI432 Ekkamai

## Objective

Create a polished mobile-first concept redesign for SHAKARIKI432 Ekkamai that presents the branch as a lively Osaka-style izakaya for late-night dining, groups, and karaoke/private rooms. The goal is to make calling, navigating, and booking feel effortless from a phone.

## Target Audience

- Thai/Japanese/expat customers around Ekkamai and Thonglor
- Groups looking for izakaya food, drinks, and private-room options
- Late-night diners searching on mobile
- People comparing restaurants quickly from maps/social links

## Current Problems Observed

1. Branch information exists but feels more like a directory/info page than a high-converting restaurant landing page.
2. Phone/map/private-room CTA can be made much more visible above the fold.
3. Value proposition can be sharper: Osaka izakaya, late-night food, 150+ dishes, karaoke rooms, group-friendly.

## Proposed Improvements

1. Cinematic dark-red/charcoal hero with strong “Osaka Night Izakaya in Ekkamai” positioning.
2. Sticky mobile CTA bar: Call, Map, Reserve.
3. Menu/value cards: Osaka bites, sushi/sashimi, drinks, karaoke rooms.
4. Clear contact block with source-backed phone/address.
5. Trust/value bullets without making unsupported claims.
6. Visible disclaimer that this is a concept redesign demo.

## Page Sections

- Hero with phone/map CTAs
- “Why go tonight” value cards
- Signature experience section
- Private rooms/karaoke section
- Location/contact section
- Footer disclaimer

## Style Direction

- Mood: premium Osaka night market meets modern Bangkok landing page
- Colors: charcoal black, lacquer red, warm lantern gold, rice-white text
- Typography: display serif-like headings, clean sans-serif body via safe CSS fallbacks
- Imagery: CSS gradients/patterns only for demo; no real brand assets
- CTA tone: direct, useful, mobile-first

## Required Disclaimer

> Concept redesign demo prepared for discussion. This is not the official website unless approved by the business owner.
