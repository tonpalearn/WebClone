# Outreach Package — Classy Clinic (สาขาราชพฤกษ์)

> โทน: ช่วยเหลือ ไม่กดดัน มืออาชีพ · ขออนุมัติจากต้นก่อนส่งจริงทุกครั้ง (มี side-effect)

## 1) ข้อความทักครั้งแรก (สั้น — Line/อีเมล/แชต)
สวัสดีครับ ทีมออกแบบเว็บไซต์ครับ ผมชอบคอนเซ็ปต์ "Premium Gold" ของ Classy Clinic มากเลยครับ เลยลองทำ **ตัวอย่างหน้าเว็บใหม่ของคลินิก (เดโม)** ให้ดูเทียบกับเว็บปัจจุบัน เน้นให้ดูพรีเมียมขึ้น โหลดเร็ว และลูกค้าบนมือถือกดโทร/จองคิวได้ง่ายขึ้นครับ

เป็นตัวอย่างเพื่อพูดคุยเท่านั้น ยังไม่ใช่เว็บทางการนะครับ ถ้าสนใจผมส่งลิงก์เดโมให้ดูได้เลยครับ ไม่มีค่าใช้จ่าย 🙏

## 2) สคริปต์โทร 30 วินาที
"สวัสดีครับ รบกวนสายแป๊บเดียวครับ — ผมทำงานออกแบบเว็บไซต์ พอดีเห็นเว็บ Classy Clinic แล้วชอบแนวพรีเมียมของคลินิกมากครับ ผมเลยลองทำหน้าเว็บใหม่เป็นตัวอย่างให้ดูเทียบกับเว็บเดิม จุดที่ปรับคือใช้ภาพจริงให้ดูน่าเชื่อถือ จัดเมนูให้ง่ายขึ้น และทำให้ลูกค้าบนมือถือกดโทรหรือจองคิวได้ทันที เป็นเดโมเพื่อให้ดูเฉย ๆ ครับ ไม่มีค่าใช้จ่าย ขออนุญาตส่งลิงก์ให้ทางไลน์ได้ไหมครับ"

## 3) Value proposition (bullets)
- **ความน่าเชื่อถือ** — ภาพแพทย์/คลินิกจริงแทน placeholder ที่ไม่ขึ้น สร้างความมั่นใจก่อนตัดสินใจ
- **ปิดการขายบนมือถือ** — ปุ่ม "ปรึกษาฟรี/จองคิว" + กดโทรทันที + แถบลอยล่างจอ
- **เมนูที่หาง่าย** — รวมเมนูซ้ำ 2 ชั้นให้เหลือชั้นเดียว จัดบริการเป็น 6 กลุ่ม
- **พรีเมียมสม่ำเสมอ** — โทนขาว-ทอง-ดำ สื่อ "Premium Gold Standard" ทั้งเว็บ
- **โหลดเร็ว** — หน้าเดียวเบา เหมาะกับยิงโฆษณา/ส่งลิงก์หาลูกค้า

## 4) Before / After (สรุปสั้น)
| | เว็บเดิม | เว็บใหม่ (เดโม) |
|---|---|---|
| รูปภาพ | placeholder ไม่ขึ้นหลายจุด | ภาพแพทย์/คลินิกจริง คมชัด |
| เมนู | ซ้ำ 2 ชั้น ยาว หายาก | ชั้นเดียว จัดกลุ่มบริการ |
| มือถือ | ต้องซูม/เลื่อน | mobile-first + แถบโทรลอย |
| ปุ่มติดต่อ | จาง กลืนพื้นหลัง | "จองคิว" เด่น กดโทรทันที |
| ภาพลักษณ์ | เทมเพลตเก่า | Premium Gold หรู |

## 6) ข้อความติดตาม (ถ้ายังไม่ตอบใน 2–3 วัน)
สวัสดีครับ รบกวนตามเรื่องเดโมเว็บ Classy Clinic ที่ส่งให้นะครับ 🙏 ถ้าได้ลองเปิดดูแล้วอยากปรับมุมไหนหรือเพิ่มบริการ/โปรไหนเป็นพิเศษ บอกได้เลยครับ ผมยินดีปรับให้ตรงกับที่คลินิกต้องการครับ
