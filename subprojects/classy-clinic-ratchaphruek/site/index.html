เมนู ✗ เมนูซ้ำซ้อนสองชั้น หัวข้อยาว หายาก ✓ เมนูเดียวกระชับ จัดบริการเป็นกลุ่ม สแกนง่าย
มือถือ ✗ ไม่ responsive ต้องซูม/เลื่อนแนวนอน ✓ mobile-first พอดีจอ + แถบโทรลอยล่าง
ปุ่มติดต่อ / Line ✗ ปุ่มจาง กลืนพื้นหลัง ไม่กดโทรบนมือถือ ✓ ปุ่ม “ปรึกษาฟรี/จองคิว” เด่น กด tel: โทรได้ทันที
ภาพลักษณ์แบรนด์ ✗ หน้าตาเทมเพลตเก่า ป้ายโปรรก ลดความพรีเมียม ✓ โทนขาว-ทอง-ดำหรู สื่อ Premium Gold Standard
