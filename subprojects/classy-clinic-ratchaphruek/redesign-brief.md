# Redesign Brief — Classy Clinic (สาขาราชพฤกษ์)

## เป้าหมาย
ยกระดับเว็บคลินิกความงามให้ **ดูพรีเมียม น่าเชื่อถือ** และ **เปลี่ยนผู้ชมเป็นการโทร/จองคิว** โดยเฉพาะบนมือถือ — สอดคล้องจุดยืนแบรนด์ "Premium Gold Standard"

## ทิศทางการออกแบบ — "Clinical Luxe / Premium Gold"
- **อารมณ์:** หรู สะอาด น่าเชื่อถือ — คลินิกพรีเมียมที่ทำให้ลูกค้ารู้สึกปลอดภัยก่อนตัดสินใจ
- **สี:** ขาว/ครีม (#fbf9f5/#f4eee3) + ดำ/charcoal (#161310/#211d18) + ทอง (#b08d50/#c9a978)
- **ฟอนต์:** Cormorant Garamond (serif หรู หัวข้อ) + Jost (sans สะอาด ป้าย/เมนู) + IBM Plex Sans Thai (ไทย refined)
- **หลักภาพ:** ภาพถ่ายจริงเป็นพระเอก (แพทย์ปรึกษา / interior คลินิก / ผิวสวย) — เลิก hero gradient text + glass + emoji
- **ทอง accent คม:** เส้น goldrule, ตัวเลข serif, ปุ่ม hover เป็นทอง

## โครงหน้า new.html (เว็บใหม่)
1. **Top nav** — โลโก้จริง + เมนูเดียวกระชับ (บริการ/เทคโนโลยี/ทำไมต้องเรา/ติดต่อ) + ปุ่มโทร · มี hamburger บนมือถือ
2. **Hero** — ภาพแพทย์จริงเต็มจอ + ken-burns, H1 serif "ความงามที่ออกแบบให้เป็นคุณ", ปุ่ม "ปรึกษาฟรี · จองคิว", แถบเวลา/ที่ตั้ง
3. **Marquee บริการ** — แถบรายชื่อหัตถการคั่นด้วย ◆ ทอง
4. **Services** — 6 กลุ่มบริการจริง การ์ดเส้นทอง hover ยก + ภาษากลาง "ปรึกษาแพทย์/วางแผนเฉพาะบุคคล"
5. **Tech** — ภาพ interior + เทคโนโลยี (Ultherapy/Thermage/HIFU) บนพื้น charcoal ทอง
6. **Trust** — ภาพผิวสวย + 4 เหตุผล (ดูแลโดยแพทย์/ปรึกษาก่อน/เครื่องมือมาตรฐาน/ใส่ใจรายละเอียด)
7. **CTA** — "พร้อมออกแบบความงามในแบบของคุณ" + ปุ่มโทร 2 สาย
8. **Visit** — ที่ตั้ง/เวลา/เบอร์จริง + ปุ่มแผนที่ บนพื้นดำ
9. **Footer + MobileBar** — disclaimer ไทย/อังกฤษ + แถบโทร/แผนที่ลอยล่างบนมือถือ

## หน้า comparison (index.html — หน้าหลักสำหรับเสนอขาย)
- Hero "เว็บเดิม vs เว็บใหม่"
- **Drag slider** ลากเทียบ old↔new ในเฟรมเดียว
- **Side-by-side** iframe คู่ (เดิม/ใหม่) + ปุ่มเปิดเต็มจอ
- **ตารางเทียบ 5 จุด:** รูปภาพ / เมนู / มือถือ / ปุ่มติดต่อ-Line / ภาพลักษณ์แบรนด์
- **กล่องข้อเสนอ** + ตัวเลขแนวทาง (เมนู 2 ชั้น→1, 1 แตะโทร, 100% mobile, รูปจริง)
- disclaimer ครบ

## หน้า old.html (จำลองเว็บเดิม)
จำลองจุดอ่อนเพื่อการเปรียบเทียบ: เมนูซ้ำ 2 ชั้น, รูป/แบนเนอร์ placeholder ไม่ขึ้น, การ์ดบริการเรียงยาวรก, ปุ่ม Line/โทรจาง, ป้าย "แบบจำลองจากเว็บเดิม • ไม่ใช่ภาพหน้าจอจริง" — **ไม่คัดลอกข้อความ/โค้ด/ดีไซน์จริง**

## ขอบเขตที่ "พอดี" (ไม่ over-engineer)
- Single-file HTML + Tailwind CDN + React UMD + Babel (ไม่มี build step)
- ยังไม่ต่อระบบจองจริง/CMS ในเดโม — โฟกัสหน้าตา + conversion path ก่อน
- เฟสถัดไป (ถ้าปิดงานได้): ฟอร์มจองจริง, หน้า service ราย treatment, ระบบรีวิว (ภายใต้ขอบเขตการแพทย์), SEO

## Compliance (ย้ำ)
- ใช้รูปจริงเพื่อเสนอแนวทางเท่านั้น · ห้ามคัดลอกข้อความ/โค้ด/ดีไซน์เว็บเขา
- ห้ามแต่งราคา/รีวิว/เคลมผลการรักษา — ใช้ "ปรึกษาแพทย์" ทุกจุด
- ทุกหน้า `noindex,nofollow` + disclaimer ไทย/อังกฤษ
