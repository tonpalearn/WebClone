# Lead Profile — Classy Clinic (สาขาราชพฤกษ์)

> ข้อมูลจากหน้าเว็บสาธารณะของธุรกิจ ใช้เพื่อจัดทำข้อเสนอ redesign เท่านั้น

## ข้อมูลธุรกิจ
- **ชื่อ:** Classy Clinic — คลินิกความงาม สาขาราชพฤกษ์
- **หมวด:** Aesthetic / Beauty Clinic (คลินิกเวชกรรมความงาม)
- **จุดยืนแบรนด์:** "Premium Gold Standard" (โทนขาว-ครีม / ดำ-charcoal / ทอง)
- **เว็บเดิม:** https://classyclinic.com/
- **ที่อยู่:** 96/9 ม.1 ต.อ้อมเกร็ด อ.ปากเกร็ด นนทบุรี 11120
- **โทร:** 062-829-3563, 091-152-4266
- **เวลาทำการ:** อังคาร–อาทิตย์ 10:00–20:00 (ปิดวันจันทร์)
- **ช่องทางติดต่อสาธารณะ:** เบอร์โทร 2 สาย + Line (ในเว็บเดิม)

## บริการ (จากเว็บสาธารณะ)
- ฟิลเลอร์ / ออกแบบรูปหน้า (Filler & Face Design)
- โบทูลินั่ม ท็อกซิน (Botulinum Toxin / โบทอกซ์)
- ร้อยไหม (Thread Lift)
- ยกกระชับด้วยพลังงาน: Ultherapy / Thermage / HIFU
- ฟื้นฟูผิว: Profhilo / Sculptra / Radiesse
- ศัลยกรรมจมูก (Rhinoplasty)

## จุดอ่อนเว็บเดิม (ที่สังเกตได้)
1. **รูปไม่ขึ้นหลายจุด** — มี SVG/placeholder แทนรูปจริงเยอะ ทำให้ดูไม่น่าเชื่อถือ ลดความพรีเมียมที่แบรนด์ชู
2. **เมนูซ้ำซ้อน** — หัวข้อบริการเรียงยาว ซ้ำกันหลายชั้น หาเมนูที่ต้องการยาก
3. **มือถือไม่ลงตัว** — เลย์เอาต์ไม่ responsive เต็มที่ ต้องซูม/เลื่อน
4. **Line/ปุ่มติดต่อไม่ชัด** — ปุ่มติดต่อ/Line กลืนกับพื้นหลัง ไม่เด่นพอให้ลูกค้ากดทันที
5. **ภาพลักษณ์** — หน้าตาโดยรวมยังไม่สื่อความเป็นคลินิกพรีเมียมเท่าที่แบรนด์วางตัว

## คุณค่าที่ redesign จะสร้าง
- ภาพแพทย์/คลินิกจริงเป็นพระเอก → สร้างความเชื่อมั่นก่อนตัดสินใจฉีด/ทำ
- โทนขาว-ทอง-ดำหรู สื่อ "Premium Gold Standard" สม่ำเสมอ
- ปุ่ม "ปรึกษาฟรี/จองคิว" + tel: เด่นทุกหน้าจอ + แถบโทรลอยล่างบนมือถือ
- จัดบริการ 6 กลุ่มที่สแกนง่าย แทนเมนูยาวซ้ำ
- โหลดเร็ว เหมาะกับการยิงโฆษณา/ส่งลิงก์

## ความเสี่ยง / ข้อควรระวัง
- เป็นธุรกิจการแพทย์ → ห้ามแต่งราคา/รีวิว/เคลมผลการรักษา ใช้ภาษากลาง "ปรึกษาแพทย์" (ทำแล้วในเดโม)
- คู่แข่งเอเจนซีอาจเข้าหาเจ้าของรายนี้อยู่แล้ว (แบรนด์มีระดับ) → ต้องเสนอเร็วและโชว์เดโมจริง
