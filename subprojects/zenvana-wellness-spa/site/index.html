หัวข้อ เว็บเดิม เว็บใหม่
ปุ่มจอง / CTA ✗ ปุ่มจองจาง กลืนพื้นหลัง ต้องมองหา ✓ ปุ่มจองคิว/โทร/ไลน์เด่นทุกจุด + แถบลอยล่างจอ
เวลาเปิด / ที่ตั้ง ✗ ฝังลึกในหน้าติดต่อ หาไม่เจอจากหน้าแรก ✓ เวลาเปิด+สาขาราชพฤกษ์เห็นชัดบนหน้าแรก
ภาพลักษณ์ / รูป ✗ รูป placeholder ไม่สื่ออารมณ์เวลเนส ✓ รูปจริงของร้านเต็มตา โทนธรรมชาติสงบ
มือถือ ✗ หน้ากว้างคงที่ ต้องซูม/เลื่อนแนวนอน ✓ mobile-first พอดีจอ ปุ่มใหญ่กดง่าย
บริการ ✗ เมนูยาวเรียงรวมกัน เลือกยาก ✓ จัดเป็น journey + Signature Retreat เด่น
