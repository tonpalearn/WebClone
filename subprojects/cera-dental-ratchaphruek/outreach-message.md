# Outreach Package: CERA Dental Clinic (คลินิกทันตกรรมราชพฤกษ์)

## Short Thai First Message

สวัสดีครับ/ค่ะ ทาง CERA Dental Clinic (คลินิกทันตกรรมราชพฤกษ์) นะครับ พอดีผมเห็นเว็บไซต์ของคลินิก เลยลองทำตัวอย่างหน้าเว็บเวอร์ชันใหม่ให้ดูเป็นคอนเซ็ปต์ครับ เน้นให้ดูสะอาด น่าเชื่อถือ ใช้ง่ายบนมือถือ บอกเวลาทำการชัดเจน และมีปุ่มโทร/ทักไลน์/นัดหมายที่กดง่ายขึ้น

ทำเป็นหน้าเปรียบเทียบ "เว็บเดิม↔เว็บใหม่" ให้เห็นภาพชัด ๆ ถ้าสะดวกผมส่งลิงก์ตัวอย่างให้ดูได้ครับ ดูแล้วถ้าชอบค่อยคุยต่อเรื่องปรับเป็นเว็บจริงของคลินิกได้ครับ ไม่มีข้อผูกมัดใด ๆ ครับ

## 30-Second Call Script

สวัสดีครับ ผมชื่อ [ชื่อผู้ติดต่อ] นะครับ ผมทำงานด้านออกแบบเว็บไซต์ให้ธุรกิจในพื้นที่ พอดีผมลองดูเว็บของ CERA Dental Clinic แล้วเห็นว่าถ้าปรับหน้าแรกให้ดูสะอาดน่าเชื่อถือขึ้น โหลดเร็วขึ้น แล้วเพิ่มเวลาทำการกับปุ่มติดต่อให้ชัด น่าจะช่วยให้คนที่เสิร์ชเจอตัดสินใจโทรหรือทักไลน์เข้ามาง่ายขึ้นครับ

ผมเลยทำตัวอย่างหน้าเว็บใหม่ไว้ให้ดูฟรี ทำเป็นหน้าเทียบกับเว็บเดิมเลย ถ้าสนใจผมส่งลิงก์ให้ดูได้ครับ เป็นแค่ตัวอย่างคอนเซ็ปต์ ยังไม่ใช่เว็บจริง ดูแล้วชอบค่อยว่ากันต่อครับ

## Value Proposition Bullets

- หน้าเว็บดูสะอาดและน่าเชื่อถือขึ้นภายในไม่กี่วินาทีแรก เหมาะกับคลินิกทันตกรรม
- บอกเวลาทำการชัดเจน (จุดที่เว็บเดิมยังไม่มี) ลดความลังเลของคนไข้ใหม่
- ปุ่มโทร/ทักไลน์/นัดหมายชัดเจน ไม่ซ้ำซ้อน บนมือถือกดง่าย
- โหลดเร็วขึ้น เหมาะกับคนที่เสิร์ชเจอบนมือถือและการยิงโฆษณา/ส่งลิงก์

## Before/After Critique Summary

| หัวข้อ | เว็บเดิม | เว็บใหม่ (คอนเซ็ปต์) |
|---|---|---|
| มือถือ | ยังไม่ชัด ปุ่ม/เลย์เอาต์ไม่ปรับตามจอ | ออกแบบมือถือมาก่อน แตะง่าย มีแถบติดต่อลอย |
| ความเร็ว | รูปเยอะ โหลดช้า | placeholder เบา โหลดไว |
| ปุ่มนัด/ติดต่อ | ปุ่ม "ปรึกษา" ซ้ำหลายจุด สับสน | หนึ่งจุดประสงค์ต่อหนึ่งปุ่ม ชัดเจน |
| ความน่าเชื่อถือ | ไม่มีเวลาทำการ ขาดข้อมูลหมอ | เวลาทำการเด่น + จุดน่าเชื่อถือ (เท่าที่มีหลักฐาน) |
| ดีไซน์ | รูปเยอะ ดูรก ไม่มีลำดับสายตา | สะอาด สว่าง โปร่ง โทนฟ้า/ขาวมืออาชีพ |
