# Redesign Brief: CERA Dental Clinic (คลินิกทันตกรรมราชพฤกษ์)

## Objective

ออกแบบคอนเซ็ปต์เว็บไซต์ใหม่แบบ "มือถือมาก่อน" (mobile-first) ให้ CERA Dental Clinic เพื่อแสดงให้เห็นว่าเว็บที่สะอาด โหลดเร็ว บอกเวลาทำการชัด และมีปุ่มติดต่อที่ไม่สับสน จะช่วยเพิ่มความน่าเชื่อถือและทำให้ลูกค้าโทร/ทักไลน์/นัดหมายเข้ามามากขึ้น โดยหน้าแรกของ demo เป็น "หน้าเปรียบเทียบเว็บเดิม↔เว็บใหม่" เพื่อให้ทีมขายใช้พูดคุยกับเจ้าของได้ทันที

## Target Audience

- คนแถวนนทบุรี/ราชพฤกษ์ที่กำลังหาคลินิกทันตกรรม
- ผู้ใช้ที่เสิร์ชบนมือถือ (กลุ่มหลัก) และต้องการโทร/ทักไลน์เร็ว
- คนที่กำลังเปรียบเทียบหลายคลินิกก่อนตัดสินใจ — ตัดสินจากความน่าเชื่อถือใน 5 วินาทีแรก
- ผู้ที่สนใจงานเฉพาะทาง (รากฟันเทียม, วีเนียร์, จัดฟันใส) ซึ่งเป็นการตัดสินใจที่ต้องการความมั่นใจสูง

## Current Problems Observed

1. **ไม่ระบุเวลาทำการ** — ลูกค้าไม่รู้ว่าเปิดกี่โมง ลดความมั่นใจและทำให้พลาดการติดต่อ
2. **ปุ่ม "ปรึกษา" ซ้ำหลายจุด** — สับสนว่าควรกดอันไหน ไม่มีลำดับการกระทำที่ชัด
3. **มือถือไม่ชัด + รูปเยอะโหลดช้า** — กลุ่มผู้ใช้หลักเจอประสบการณ์ที่หนักและช้า ฟอร์มติดต่ออยู่ลึกต้องเลื่อนหานาน ขาดข้อมูลทันตแพทย์/จุดน่าเชื่อถือที่ดูเป็นมืออาชีพ

## Proposed Improvements

1. Hero ชัดเจน บอกว่าเป็นใคร อยู่ที่ไหน ทำอะไรได้ พร้อมปุ่มโทร/Line/นัดหมายตั้งแต่หน้าจอแรก
2. ปุ่มติดต่อหลักเด่นและไม่ซ้ำซ้อน — หนึ่งจุดประสงค์ต่อหนึ่งปุ่ม (โทร / ทักไลน์ / นัดหมาย) มีแถบติดต่อลอยล่างจอบนมือถือ
3. ส่วนบริการจัดกลุ่มชัด (รากฟันเทียม · จัดฟัน · วีเนียร์/ครอบฟัน · ทันตกรรมทั่วไป · ทันตกรรมเด็ก) สแกนง่ายบนมือถือ
4. **เวลาทำการแสดงเด่น** — แก้จุดอ่อนใหญ่สุด (ใช้ค่า placeholder กำกับให้ยืนยันกับคลินิก)
5. จุดน่าเชื่อถือเฉพาะที่อ้างอิงได้ (ทันตแพทย์เฉพาะทาง, เครื่องมือทันสมัย, ทำเลราชพฤกษ์) — ไม่มีรีวิว/รางวัล/ราคาที่แต่งขึ้น
6. แผนที่/ติดต่อชัดเจน (ที่อยู่ + เบอร์จริง + Line จริง) และเลย์เอาต์มือถือมาก่อน โหลดเร็วด้วย placeholder แทนรูปหนัก

## Page Sections

- Hero (ชื่อคลินิก + สิ่งที่ทำ + ปุ่มโทร/Line/นัดหมาย)
- บริการหลัก (จัดกลุ่ม)
- เวลาทำการ (เด่น — แก้จุดอ่อนเดิม)
- ทำไมเลือกที่นี่ / จุดน่าเชื่อถือ (เฉพาะที่มีหลักฐาน)
- แกลเลอรี (placeholder — ไม่ใช้รูปจริง)
- ติดต่อ / แผนที่ / Location
- Footer + disclaimer

## Style Direction

- Mood: สะอาด สว่าง น่าเชื่อถือ มืออาชีพ โปร่ง (เหมาะกับทันตกรรม) — glass premium บนพื้นสว่าง
- Colors: ฟ้า/เทอร์ควอยซ์/ขาว (สี clinical สื่อความสะอาดและความเชื่อใจ) — เน้นพื้นที่ว่าง ไม่อึดอัด
- Typography: ฟอนต์อ่านง่าย คมชัด หัวเรื่องหนักแน่นแต่ไม่แข็ง
- Imagery: placeholder ทั้งหมด (ไอคอน/บล็อกสีไล่เฉด/SVG) — ไม่ใช้รูปหรือโลโก้จริงของคลินิก
- CTA tone: ชวนปรึกษาแบบสุภาพ ไม่กดดัน ใช้ภาษากลาง "ปรึกษาทันตแพทย์" ไม่เคลมผลการรักษา

## Medical Compliance

- ไม่เคลมผลการรักษา ไม่รับประกันผล ไม่ระบุราคา
- ไม่ใส่รีวิว/รางวัล/credential ที่ไม่มีหลักฐานสาธารณะ
- ใช้ภาษากลาง "ปรึกษาทันตแพทย์" / "สอบถามรายละเอียดเพิ่มเติม"
- เวลาทำการเป็น placeholder กำกับว่า "โปรดยืนยันเวลาจริงกับคลินิก"
- ไม่ใช้โลโก้/รูปจริง — placeholder ทั้งหมด
