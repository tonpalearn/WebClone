# Redesign Brief: บ้านปัญญา เฟอร์นิเจอร์ไม้สัก (BNP Furniture)

## Objective

ทำคอนเซ็ปต์ redesign แบบ mobile-first ให้ร้านบ้านปัญญา เฟอร์นิเจอร์ไม้สัก ดูเป็นแบรนด์งานคราฟต์ไม้สักแท้ที่อบอุ่น หรู มีระดับ — ให้ลูกค้าที่เปิดจากมือถือเห็นของสวย เข้าใจจุดขาย "ไม้สักแท้ + งานสั่งทำ" และโทร/แชต Line ได้ทันที

## Target Audience

- ลูกค้าที่กำลังหาเฟอร์นิเจอร์ไม้สักแท้ (โต๊ะหมู่บูชา ตู้พระ เตียง โต๊ะอาหาร โซฟา)
- คนที่ต้องการงานสั่งทำตามแบบ/ขนาดบ้าน
- ลูกค้าย่านนนทบุรี/ราชพฤกษ์/กรุงเทพฝั่งตะวันตก ที่ค้นหาบนมือถือ
- คนที่เทียบหลายร้านอย่างรวดเร็วจากแผนที่/โซเชียล

## Current Problems Observed

1. เว็บเดิมเป็นเทมเพลตสำเร็จรูป LnwShop รุ่นเก่า ไม่ responsive — บนมือถือต้องซูม/เลื่อนแนวนอน หน้าตาไม่สื่อความเป็นไม้สักแท้ระดับพรีเมียม
2. หมวดสินค้า 19 หมวดเรียงยาวรกตา ไม่จัดกลุ่มให้สแกนง่าย + ปุ่มติดต่อ/โทร/Line จาง กลืนพื้นหลัง ลูกค้าต้องมองหา
3. โหลดช้า (วิดเจ็ตสำเร็จรูปเยอะ) และ typography ไม่เป็นระบบ (ขนาด/ฟอนต์ปนกัน) ทำให้ภาพรวมดูไม่น่าเชื่อถือเท่าคุณภาพงานจริง

## Proposed Improvements

1. Hero งานไม้สักโทนอบอุ่น (น้ำตาลไม้/ทอง/ครีม) พร้อมพาดหัวจุดขาย "ไม้สักแท้ · งานสั่งทำ" ชัดเจน
2. ปุ่ม CTA เด่น: โทรเลย / แชต Line @teak — ทั้งบนหัวเว็บและแถบลอยล่างจอบนมือถือ (1 แตะ)
3. จัดหมวดสินค้าให้น้อยลงเป็นกลุ่มชัด (เช่น งานบูชา/ห้องนอน/ห้องนั่งเล่น/ห้องอาหาร/งานสั่งทำ) แทน 19 หมวดเรียงยาว
4. บล็อกจุดเด่น/ความน่าเชื่อถือเฉพาะที่มีหลักฐาน: ไม้สักแท้, งานสั่งทำ, ทำเลริมถนนราชพฤกษ์
5. แกลเลอรี placeholder (gradient/ไอคอน) สื่อหมวดงาน โดยไม่ใช้รูปจริงของร้าน
6. บล็อกแผนที่/ติดต่อชัดเจน พร้อมที่อยู่+เบอร์+Line จริง และ disclaimer ว่าเป็นคอนเซ็ปต์เดโม

## Page Sections

- Hero พร้อมปุ่มโทร/Line
- จุดขายไม้สักแท้ (value strip)
- หมวดสินค้าเด่น (จัดกลุ่ม)
- จุดเด่น/ทำไมต้องบ้านปัญญา
- แกลเลอรี placeholder
- แผนที่/ติดต่อ (ริมราชพฤกษ์)
- Footer disclaimer

## Style Direction

- Mood: อบอุ่น หรู งานคราฟต์ไม้สัก — เหมือน showroom เฟอร์นิเจอร์พรีเมียมมาเป็น landing page
- Colors: น้ำตาลไม้สัก (teak/walnut), ทองอุ่น (brass/gold), ครีม/งาช้าง, แต้มเขียวมรกตได้เล็กน้อย, พื้นเข้มหรู
- Typography: หัวเรื่อง serif-like ดูคลาสสิกมีระดับ, เนื้อหา sans-serif อ่านง่าย (ใช้ CSS fallback ปลอดภัย)
- Imagery: gradient/ไอคอน/ลายไม้แบบ CSS เท่านั้นสำหรับเดโม — ไม่ใช้ภาพจริง/โลโก้จริงของร้าน
- CTA tone: สุภาพ ตรงประเด็น ใช้ง่ายบนมือถือ
