# Outreach Package: บ้านปัญญา เฟอร์นิเจอร์ไม้สัก (BNP Furniture)

## Short Thai First Message

สวัสดีครับ ผมเห็นเว็บไซต์ของร้านบ้านปัญญา เฟอร์นิเจอร์ไม้สัก แล้วชอบงานไม้สักของทางร้านมากครับ พอดีผมลองทำตัวอย่างหน้าเว็บเวอร์ชันใหม่ให้ดูเป็นคอนเซ็ปต์ เน้นให้ดูอบอุ่น มีระดับ สมกับงานไม้สักแท้ ใช้ง่ายบนมือถือ และมีปุ่มโทร/แชต Line ที่ชัดเจนขึ้นครับ

ถ้าสะดวก ผมส่งลิงก์ตัวอย่างให้ดูได้ครับ ไม่มีค่าใช้จ่าย ถ้าชอบค่อยคุยต่อเรื่องปรับเป็นเว็บจริงของร้านได้ครับ

## 30-Second Call Script

สวัสดีครับ ผมชื่อ [ชื่อผู้ติดต่อ] นะครับ ผมทำงานด้านออกแบบเว็บไซต์ให้ธุรกิจท้องถิ่น พอดีผมลองดูเว็บของร้านบ้านปัญญา เฟอร์นิเจอร์ไม้สัก แล้วเห็นว่าถ้าปรับหน้าแรกให้ดูพรีเมียมขึ้น จัดหมวดสินค้าให้ดูง่ายขึ้น และใช้งานบนมือถือได้ดีขึ้น น่าจะช่วยให้ลูกค้าติดต่อ/สอบถามงานสั่งทำง่ายขึ้นครับ ผมเลยทำตัวอย่างสั้น ๆ ไว้ให้ดูฟรี ถ้าสนใจผมส่งให้ดูได้ครับ

## Value Proposition Bullets

- หน้าเว็บดูอบอุ่น มีระดับ สมกับงานไม้สักแท้ภายในไม่กี่วินาทีแรก
- ลูกค้าบนมือถือโทร/แชต Line ได้ทันที ปุ่มเด่น กดง่าย
- จัดหมวดสินค้าจาก 19 หมวดให้เหลือเป็นกลุ่มที่สแกนง่าย หาของเจอเร็ว
- เว็บโหลดเร็วขึ้น เหมาะกับการยิงโฆษณาหรือส่งลิงก์ให้ลูกค้า
