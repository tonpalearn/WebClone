# ราชพฤกษ์ นวดเพื่อสุขภาพแอนด์สปา — Redesign Demo

คอนเซ็ปต์ออกแบบเว็บใหม่สำหรับร้าน **ราชพฤกษ์ นวดเพื่อสุขภาพแอนด์สปา** (ปากเกร็ด นนทบุรี)
เพื่อใช้เสนอขายงานออกแบบเว็บไซต์ — **เป็นตัวอย่างเพื่อพูดคุย ไม่ใช่เว็บทางการ**

## ไฟล์ในโฟลเดอร์
| ไฟล์ | คืออะไร |
|---|---|
| `lead-profile.md` | โปรไฟล์ลีด + คะแนนคุณสมบัติ (24/25) |
| `redesign-brief.md` | บรีฟทิศทางออกแบบ "Thai Warm Wellness" + สี/ฟอนต์/โครงหน้า |
| `outreach-message.md` | ข้อความทัก + สคริปต์โทร + before/after |
| `site/index.html` | **หน้าหลัก = หน้าเทียบเว็บเดิม↔ใหม่** (slider + side-by-side + ตาราง 5 จุด) |
| `site/new.html` | เว็บใหม่ (React + Tailwind + รูปจริง โทนไม้อุ่น) |
| `site/old.html` | จำลองเว็บเดิม (สื่อจุดอ่อน มีป้าย "จำลอง") |
| `assets/` | รูปนวดจริง 5 รูป + โลโก้ (ดูดจาก rachamassage.com) |

## วิธีเปิดดู
เปิดไฟล์ในเบราว์เซอร์ผ่าน local server (เพราะ new.html ใช้ iframe + React CDN):
```bash
cd subprojects/rachaphruek-massage-spa/site
python3 -m http.server 8080
# แล้วเปิด http://localhost:8080/index.html
```
หรือดับเบิลคลิก `site/index.html` ได้เลย (iframe ใช้ path relative)

## สแต็ก
HTML single-file ต่อหน้า · Tailwind CDN · React 18 UMD + Babel (classic preset, pin `@babel/standalone@7.23.6`) · Google Fonts (Trirong + IBM Plex Sans Thai)

## รูปจริง
ดูดมา **5 รูป** จาก `rachamassage.com` (อนุมัติแล้ว) — `massage-01/03/07/08/09.webp` + `logo.svg`
ตัด `massage-02` ออก (โคลสอัพหน้าท้อง ใกล้ชิดเกินไป ไม่เหมาะ family-friendly)

## ข้อควรรู้ (compliance)
- ทุกหน้าใส่ `noindex,nofollow` + disclaimer ภาษาไทย
- ใช้รูปจริงประกอบการนำเสนอได้ (อนุมัติแล้ว) แต่**ไม่คัดลอกข้อความ/โค้ด/ดีไซน์**ของเว็บเดิม
- **ไม่แต่งราคา** — ตารางราคาใช้ปุ่ม "สอบถามราคา" เพราะเว็บเดิมไม่มีราคา
- ไม่กุรีวิว/ใบรับรอง — ข้อมูลพนักงาน/ใบรับรองอ้างอิงตามเว็บเดิม (ใส่ * กำกับ)
- ปุ่มไลน์เป็น placeholder (เว็บเดิมไม่มี) — เจ้าของใส่ไอดีจริงภายหลัง
