# Outreach Package — ราชพฤกษ์ นวดเพื่อสุขภาพแอนด์สปา

## ข้อความทักครั้งแรก (สั้น, ไม่กดดัน)
> สวัสดีครับ ทางร้านราชพฤกษ์ นวดเพื่อสุขภาพแอนด์สปา ผมชอบบรรยากาศและบริการของร้านมากครับ เลยลองทำตัวอย่างหน้าเว็บใหม่ให้ดูเล่น ๆ — เพิ่มเมนูบริการให้ชัด มีปุ่มโทร/ไลน์/แผนที่ที่ลูกค้ากดจองได้ทันที และทำให้ดูสวยบนมือถือ
>
> เป็นแค่ตัวอย่างเพื่อพูดคุยนะครับ ไม่ใช่เว็บจริง ถ้าสนใจผมส่งลิงก์ให้ดูเทียบกับเว็บเดิมได้เลยครับ ลองดูแล้วไม่ถูกใจก็ไม่เป็นไรครับ

## สคริปต์โทร 30 วินาที
> "สวัสดีครับ รบกวนไม่นานนะครับ — ผมทำเว็บไซต์ให้ร้านนวด/สปาในย่านราชพฤกษ์ครับ
> ผมเห็นเว็บของร้านราชพฤกษ์แล้วลองทำหน้าตัวอย่างใหม่ให้ดูฟรีครับ จุดที่ปรับคือ ใส่เมนูบริการกับเวลานวดให้ชัด เพิ่มปุ่มโทรกับไลน์ให้ลูกค้าจองง่ายขึ้น และทำให้เปิดบนมือถือสวย
> ผมขออนุญาตส่งลิงก์ตัวอย่างให้ดูเทียบกับเว็บเดิมได้ไหมครับ ดูแล้วชอบค่อยคุยกันต่อ ไม่ชอบก็ไม่เป็นไรเลยครับ"

## คุณค่าที่เสนอ (bullets)
- **เห็นบริการ + เวลา ครบในหน้าเดียว** — ลูกค้าไม่ต้องโทรถามทีละอย่าง
- **ปุ่มโทร / ไลน์ / จอง เด่น + แถบลอยล่างจอบนมือถือ** — กดจองได้ใน 1 แตะ
- **ที่ตั้ง + แผนที่ชัด** — ลูกค้าหาทางมาง่าย ลดการถามเส้นทาง
- **โทนแบรนด์อุ่น ผ่อนคลาย** — สื่อความเป็นมืออาชีพ ชวนให้รู้สึกอยากมา
- **โครงตารางราคาพร้อมไว้** — เจ้าของเติมราคาจริงได้ทันทีเมื่อพร้อม

## สรุป Before / After
| | เว็บเดิม | เว็บใหม่ (คอนเซ็ปต์) |
|---|---|---|
| ราคา/แพ็กเกจ | ไม่มีเลย ต้องโทรถามทุกอย่าง | ตารางบริการชัด + ช่องราคา "สอบถาม" พร้อมเติมจริง |
| ช่องทางจอง | มีแต่เบอร์โทร | โทร + ไลน์ + ปุ่มจอง + แถบลอยมือถือ |
| สาขา | เอ่ยชื่อ 3 สาขา แต่ไม่มีที่อยู่/แผนที่ | บอกชัดว่ามีหลายสาขา + ปุ่มแผนที่ + โทรเช็กสาขา |
| แบรนด์/มือถือ | โทนอ่อน ปุ่มไม่เด่น | โทนไม้อุ่น+เขียวสาคู รูปจริงเป็นพระเอก mobile-first |

## ข้อความติดตาม (ถ้ายังไม่ตอบ)
> สวัสดีครับ ส่งลิงก์ตัวอย่างหน้าเว็บใหม่ของร้านมาให้ดูอีกครั้งนะครับ เผื่อพอมีเวลาลองเปิดดู — กดปุ่มโทร/ไลน์/แผนที่ในตัวอย่างได้เลยครับ ถ้ามีตรงไหนอยากปรับ บอกได้เลย ยินดีปรับให้ตรงใจร้านครับ

**โทน:** ช่วยเหลือ ไม่ตื๊อ มืออาชีพ — ทุกข้อความย้ำว่าเป็นตัวอย่างเพื่อพูดคุย ไม่ใช่เว็บทางการ
