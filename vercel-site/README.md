# TPL Design Vercel Site

Single Vercel project for WebClone demos.

Target domain/project:

```text
tpldesign.vercel.app
```

Routes:

```text
/                              landing page
/subproject/shakariki432-ekkamai SHAKARIKI432 Ekkamai demo
```

Build:

```bash
npm run build
```

Preview:

```bash
npm run preview
```

Deploy:

```bash
vercel --prod
```
